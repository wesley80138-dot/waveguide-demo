空間均勻度: 4mm pupil 掃 eyebox 9 個位置 (邊緣貼齊不超出), 每格 = 該位置所有 FOV 加總.
百分比 = 該格 / 最亮格. 主指標 mean/max = 平均亮度與最亮的比 (含 0 格). min/max 易因有 0 格而失真, Michelson = (max−min)/(max+min) 為 0 表完美均勻.

方法 (Zhang et al., Opt. Express 33, 34568, 2025):
每個 FoV 各算 BTM, 得 ηi (Eq. 3, 到 eyebox 的能量比) 與 Ui (Eq. 4, 該 FoV 在 eyebox 中卷積後 Imin/Imax).
角度均勻度 = min(ηi)/max(ηi) — 9 個 FoV 之間的亮度差.
最差空間 U = min(Ui) — 各 FoV 中眼位移動時最不均勻者.
